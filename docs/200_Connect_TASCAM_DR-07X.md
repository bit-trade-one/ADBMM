# 配線図(単体)  

<img src="images/50b803277f27e25c33dcecc3b4afb18f081f29955e03e10f4335279f712f3e18.jpg" alt="DR-07X接続"  Width="800">



- 美耳本体

- (別売)①赤白RCA - 3極 オーディオミニプラグ ケーブル
